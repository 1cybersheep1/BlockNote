## Changing the Block Side Menu

You can change or replace the Block Side Menu with your own React component. In the demo below, the button to add a new block is replaced with one to remove the hovered block.

The default Block Side Menu is replaced with the `CustomSideMenu` component. It's passed as a prop to `SideMenuController`, which controls where it's positioned and when it opens.

Passing `sideMenu={false}` to `BlockNoteView` tells BlockNote not to show the default Block Side Menu. Adding the `SideMenuController` as a child of `BlockNoteView` tells BlockNote to show the custom one instead.

Head to [Components](/docs/ui-components/side-menu#components) and [Hooks](TODO) to find out more about the contents of `CustomSideMenu` and `CustomButton`.

## Changing Drag Handle Menu Items

You can change the items in the Drag Handle Menu. The demo below adds an item that resets the block type to a paragraph.

<Example name="ui-components/side-menu-drag-handle-items" />

By using the `SideMenu` component and setting its `dragHandleMenu` prop, you create a modified Block Side Menu with your custom Drag Handle Menu. You can then replace the default Block Side Menu with your modified one as in the [previous demo](/docs/ui-components/side-menu#changing-the-side-menu).

## Components

If you want to change the default Block Side Menu or Drag Handle Menu, BlockNote provides React components you can use, which match the default styling.

### Default Components

BlockNote exports all components used to create the default layouts - both the menus themselves and the items in them. Head to the [default Side Menu's source code](https://github.com/TypeCellOS/BlockNote/blob/main/packages/react/src/components/SideMenu/mantine/SideMenu.tsx) and [default Drag Handle Menu's source code](https://github.com/TypeCellOS/BlockNote/blob/main/packages/react/src/components/SideMenu/DragHandleMenu/mantine/DragHandleMenu.tsx) to see all the components that you can use.

### Custom Components

BlockNote also provides components that you can use to make your own menu items, which also match the default styling:

```typescript
type SideMenuButtonProps = {
  children: JSX.Element;
}
export const SideMenuButton = (props: SideMenuButtonProps) => ...;

// Contains all props that a regular button element would take, as well as all props from the Mantine `Menu.Item` component.
type DragHandleMenuItemProps = PolymorphicComponentProps<"button"> & MenuItemProps
export const DragHandleMenuItem = (props: DragHandleMenuItemProps) => ...;
```
